# 🎓 StudentOS v5

StudentOS v5 is a smart student productivity dashboard built using React + Vite.

## 🚀 Live Demo
https://studentos-v5.vercel.app/

## ✨ Features
- Dashboard overview
- Task Manager
- Quick Notes
- Study Tracker
- Attendance Tracker
- Goals Section
- Pomodoro Timer
- Clean responsive UI
- Works on laptop and mobile

## 🛠️ Tech Stack
- React.js
- Vite
- CSS
- Vercel Deployment
